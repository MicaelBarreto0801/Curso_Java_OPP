☕ Jornada Java Developer: Do Zero ao Spring Boot

Este repositório é o meu diário de bordo rumo ao domínio completo do ecossistema Java. O foco é construir uma base teórica inabalável para criar aplicações modernas, escaláveis e testáveis, utilizando as versões mais recentes da stack (Java e Spring Boot).

🎯 Objetivos do Aprendizado

Neste projeto, documento minha evolução nos seguintes pilares: Domínio da OO: Aplicação prática de Abstração, Encapsulamento, Herança e Polimorfismo para resolver problemas reais. Modelagem de Sistemas: Tradução de requisitos em Diagramas de Classe UML (Entidades e Serviços).

Arquitetura de Dados: Manipulação de bancos relacionais via JDBC (SQL) e ORM (JPA/Hibernate). Integração com bancos de dados NoSQL.

Interfaces e Web Services: Desenvolvimento de aplicações Desktop com interface gráfica. Construção de APIs REST robustas seguindo as melhores práticas de mercado.

🛠️ Stack Tecnológica

Linguagem: Java 21 (LTS) Framework: Spring Boot (focado em alta performance e escalabilidade) Persistência: JPA / Hibernate / JDBC Bancos de Dados: Relacionais (SQL) e NoSQL Modelagem: UML 2.0
